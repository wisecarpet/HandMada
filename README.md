# HandMada — Art on Candles & Cakes 🎨🕯️🍰

A premium multilingual landing page for an artisan brand specializing in hand-painted candles and custom cakes.

## ✨ The Project
This isn't just a website; it’s a digital gallery for unique, handcrafted pieces. As a professional marine artist, I’ve combined fine art with web development to create a space where "every stroke has a soul."

### Key Features:
- **Multilingual Support**: Fully localized in **Russian, English, Ukrainian, and Finnish** using JSON-based data management.
- **Dynamic Galleries**: Smooth, responsive image galleries showcasing hand-painted details.
- **Clean Architecture**: Built with vanilla JavaScript, HTML5, and CSS3. No heavy frameworks — just pure, fast code.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3, JavaScript (ES6+)
- **Data Management**: JSON for content and translations.
- **Design**: Custom branding and UI/UX by the author.

## 🇫🇮 Suomeksi / 🇷🇺 На русском
The project reflects a philosophy of "Individualism in a mass-produced world," featuring artisan techniques like 3D textures on wax and edible paintings on cakes.
